What are the different ways to select an element in the DOM?
  • By Tag Name

    const element = document.getElementsByTagName("h1"); This will return all the elements of h1 tags

  • By Class Name

    const element = document.getElementsByClassName("btn"); This will return all the elements having btn class

  • By ID Name

    const element = document.getElementById("my-btn"); This will return the element having the id my-btn. Normally we just have one unique id in the total html.

  • With QuerySelector

    This is a versatile one to get either the first or all the element by ID, class or tag

    const element1 = document.querySelector(".my-class"); // to get first element having my-class
    const elements2 = document.querySelectorAll(".my-class"); // to get all the elements having my-class
    const element3 = document.querySelector("#myID"); // to get the element having myID
    const element4 = document.querySelector("p"); // returns the first p tag element
    const elements5 = document.querySelectorAll("p"); // returns all the p tag elements     pretty much versatile method for DOM

What is event delegation in the DOM?
When we have to do a repeatitive task on different elements of a same parent element we have a brute force solution. We can add event listener in all the elements and do the job. But it is not too much efficient. To solve this repeatitive Task we may select the parent element and then add the event listener in that element. Then with the help of conditionals we can simply perform the task.
Also with the brute force method we cannot add event listener in the newly added elements. But with 2nd method which is also known as event delegation we can do the same task with any children element of the parent regarless they are old or new.
What is event bubbling in the DOM?
Event bubbling is the process where the parent element event triggers upon triggering the child element. This flow starts from children and moves up to window ( the root of the DOM ).
How do you create, add, and remove elements using JavaScript?
  • Create element

    const element = document.createElement("p"); // creates a new element

  • Add element

    parentElement.appendChild(childElement)

  • Remove element

    parentElement.removeChild(childElement);

    parentElement.innerHTML = "" // this will delete all the child elements (Actually it replaces the child element with nothing. )
